K-Debloater

A debloat tool for KDE Plasma 6.

Download from Github

Installation

Download the script from the Github repo and run the following commands:

                $ chmod +x /path/to/file/kdebloater(version).sh
                # sudo ./path/to/file/kdebloater(version).sh

Replace (version) with 4deb or 4opensuse depending on if you use debian or openSUSE respectively.

Replace /path/to/file with the directory the script is downloaded in.
